<img width="1913" height="1076" alt="image" src="https://github.com/user-attachments/assets/351183d7-c6dd-4242-8ce6-402ecc9f78c9" /># 🎬 Movie Recommender System

This is a Machine Learning-based Movie Recommender System built using Python and Streamlit.

## 🚀 Features
- Recommends movies based on similarity
- Uses content-based filtering
- Fast recommendations using cosine similarity

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📂 Files
- app.py → Streamlit app
- Main.ipynb → Model building
- movies.pkl → Processed data
- top10K-TMDB-movies.csv → Dataset
- 
## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
